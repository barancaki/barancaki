# Tech Context

## Core Stack
- **Framework**: React 19 (via Vite)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **Icons**: Lucide React

## Development Environment
- **Package Manager**: npm
- **Build Tool**: Vite (fast HMR)
- **Timezone**: User is in +03:00 GMT.

## Key Dependencies
- `lucide-react`: For all iconography.
- `react-router-dom`: For client-side routing.
- `tailwindcss`: For all styling.

## Design Constraints
- **Theme**: Supports both Dark Mode (default) and Light Mode. System preference is respected but overridden by user toggle.
- **Layout**: "Bento Grid" evolved into a generic list of expandable project cards to house complex visualizers.
- **Responsive**: Complex visualizers are hidden by default, improving initial mobile load and scanability.
