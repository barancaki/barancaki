# Project Brief

## Overview
Baran Çakı's personal portfolio website, designed to showcase his expertise as an AI Engineer and Automation Architect. The site emphasizes "Vibe Coding" and features a premium, modern aesthetic using glassmorphism and neon accents.

## Core Goals
- **Showcase Expertise**: Highlight skills in n8n, Python, AI/LLM integration, and automation.
- **Visual Impact**: Use a high-quality, "dark mode" design with rich animations to impress visitors immediately.
- **Demonstrate Value**: Present complex workflows (like LinkedIn Lead Gen) in an understandable, visual way to potential clients.
- **Lead Generation**: Encourage contact through clear CTAs ("Let's Talk").

## Key Features
- **Hero Section**: Dynamic typing effect, "Vibe Coding" status, and clear value proposition.
- **Bento Grid**: A modern grid layout to display selected works and the tech stack.
- **Workflow Visualizer**: An interactive, animated diagram demonstrating a complex n8n workflow.
- **Case Study Pages**: Detailed breakdowns of specific projects (e.g., LinkedIn Lead Gen) without revealing trade secrets.
- **Tech Stack**: Visual representation of tools used.
